# BATCH91 TEACHER WORKFLOW GUARD

Batch91 focuses on a data-paused UIUX path. The academic dataset is intentionally not expanded in this batch, so the product must stay useful by making the teacher workflow clearer and safer.

## What changed
- mobile step cue for the active tab;
- next-step button on mobile;
- data-paused guard inside the compose form;
- empty editor state before lesson content exists;
- settings note that recommends Easy/Standard while data is paused;
- safer saved-lesson fallback defaults that do not silently return to grade 6/Canh Dieu;
- bottom navigation safe-area padding.

## Guardrails
- No new academic data was added.
- No seed/scaffold content was upgraded to reviewed or verified.
- No AI, model SDK, or agent feature was added.
- The UI must keep saying khung an toàn when data is not reviewed.

## Verification limits
This is a source-level UIUX batch. It does not prove Vercel build, hosted URL smoke, mobile browser QA, DOCX/PDF export runtime, or teacher feedback.
